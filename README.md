# Student Management System (Python + SQLite)

A student management system developed in Python with Flask and a SQLite database
This project allows managing students, classes, subjects, enrollments, and academic results

## Features
- Student management (add, edit, delete)
- Class management
- Subject management by class
- Student enrollment in a class
- Grade management
- Calculation of averages per student and per class

## Technologies used
- Python
- Flask
- SQLite
- HTML / CSS


## Project objective
This project aims to practice: 
- modeling relational databases
- backend development with Python
- managing academic data in a real system
